# Users & Groups

## Management

**Settings → Users**: create users, set passwords, assign groups
**Settings → Groups**: group tabs: Apps · Media · Widgets · Docker · Background
Visibility is individually configurable per group.

## Built-in Groups

| Group | Description |
|---|---|
| Admin | Full access, cannot be deleted |
| Guest | Read-only access, no Docker |

## Guest Mode for Admins

**Topbar → "Switch to Guest View"**
→ Configure the dashboard as guests will see it.
