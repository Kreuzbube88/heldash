# Design & Settings

## Design Tab (admins only)

**Settings → Design** — changes apply globally to all users.

| Setting | Options | Description |
|---|---|---|
| Corner Style | Sharp / Default / Rounded | Radius of all cards and elements |
| Background Blur | Subtle / Medium / Strong | Intensity of the glass morphism effect |
| Spacing | Compact / Comfortable / Spacious | Padding and spacing in the layout |
| Sidebar Style | Default / Minimal / Floating | Appearance of the navigation |
| Animations | Full / Reduced / None | Transitions and animations |
| Custom CSS | Free-text field | Global CSS overrides |

## Themes

**Topbar → Moon/Sun icon** → switch between Light / Dark
**Topbar → Color wheel** → accent color: Cyan / Orange / Magenta

### Auto-Theme (Settings → General)

Automatic switching by time of day. E.g. Light from 08:00, Dark from 20:00.

## Background Images

**Settings → Design → Background Images**

1. Upload image (PNG/JPG/SVG/WebP, max. 5 MB)
2. Assign under **Settings → Groups → Group → Tab "Background"**

Each group can have its own background image.
